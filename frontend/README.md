# Tech Forge Frontend

Welcome to the Tech Forge frontend application! This project is built using React and TypeScript, providing a modern and efficient user interface.

## Getting Started

To get started with the frontend application, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd tech-forge/frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```

The application will be available at `http://localhost:3000`.

## Project Structure

- `src/`: Contains the source code for the frontend application.
  - `App.tsx`: The main component that renders the application.
  - `index.tsx`: The entry point of the React application.
  - `components/`: Contains reusable components.
    - `Welcome.tsx`: Displays the welcome message.
  - `styles/`: Contains the CSS styles for the application.
